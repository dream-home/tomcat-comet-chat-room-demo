## Getting Started
用Tomcat Maven插件运行Demo：
```
git clone https://github.com/wucao/tomcat-comet-chat-room-demo.git
cd tomcat-comet-chat-room-demo
mvn tomcat7:run
```
启动成功后可以用浏览器打开http://localhost:8080/来测试聊天室。
